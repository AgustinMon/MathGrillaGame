# desarrolla una app para android y web en flutter.
la app consiste en un "crucigrama" pero en vez de letras son operaciones matemáticas simples.
el objetivo es resolver las operaciones matemáticas.
el usuario tendrá un tiempo límite para resolver las operaciones.
el usuario tendrá un contador de puntos.
el usuario tendrá un contador de vidas.
el usuario tendrá un contador de niveles.
el usuario tendrá un contador de pistas.
el usuario tendrá un contador de bonificaciones.
el usuario tendrá un contador de penalizaciones.

# UI
La aplicacion explicará al inicio como se juege.
Todos los movimientos y numeros que hay que rellener en la grilla estarán en el footer como pequeñas cajas que se deslizarán y se colocarán en su sitio
Cuando el numero u operacion sea deslizado en la casilla correcta, se resaltará la fila o columna que se haya logrado resolver
Todos los iconos, ayudas y movimientos serán animados
Desarrollo minimalista pero elegante y moderno.
Los niveles agrandan la cuadricula

Algoritmo de niveles
Los niveles deben desarrollarse en forma procedural y ser infinitos.
Las cifras a resolver no deben ser de mas de 3 cifras.
Se acepta suma, resta, division, multiplicacion.
En niveles más avanzados puede aceptarse potencia, raiz (haz la prueba con uno para evaluar si es jugable)
